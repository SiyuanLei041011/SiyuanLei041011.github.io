安装

Python解释器的安装

进入Python官网, 找到需要的版本, 点击Download Windows installer(64-bit), 安装完成后, 点击exe文件, 注意要勾选上Add Python to PATH和Install Python for all users.

我下载的版本是Python 3.13, 放在C盘根目录下, 也就是文件夹Python313, 如果需要多版本共存, 那么下载的其他版本也要放在C盘根目录下, 比如Python311, Python310等.

如果忘记勾选添加环境变量, 可以自己手动修改: 右键点击"此电脑", 点击"属性", 点击"高级系统设置", 点击"环境变量", 找到系统变量里面的PATH, 打开后添加两个路径C:\Python313和C:\Python313\Scripts\, 对于其他的版本也这样操作. 设置完成后, 可以按下Win+R, 输入cmd, 输入Python --version就可以看到当前的版本信息了. 输入python, 就可以运行Python了.

如果要切换多个版本的解释器, 那么就C:\Python313目录下, 添加一个Python.exe的副本, 命名为Python313.exe, 对于其他版本也这样操作. 然后在命令行窗口, 输入Python313/Python311/Python310 --version, 就可以分别看见不同版本的信息了.

VS Code + Python配置

在插件栏搜索Python, 直接安装Python插件.

下载好之后, 可以新建一个py文件, 在右下角可以看到解释器相应的版本, 点击可以切换. 写好代码后点击右上角的三角符号就可以运行了.

运行时, 我们可以看到运行的终端是powershell, 点击上面的小加号旁边的箭头, 可以切换运行终端为Git bash, Command Prompt等其他终端, 然后把之前的powershell删掉即可.

代码规范

代码的规范化, 搜索Python PEPs, 就可以看到相应的规范文档了, 一般情况下, 我们遵守PEP8. 在插件栏中搜索autopep8, 安装好这个插件后, 点击"查看", 点击"命令面板", 输入"format document", 就可以自动把代码规范化了.

调试

将.py文件放入一个文件夹中, 比如Python Code, 然后把这个文件放进VS Code的工作区. 在VS Code的左边一栏点击Debug选项, 点击齿轮形状的选项, 然后就会出现一个launch.json的文件, 关闭它, 然后就可以开始Debug了.

先设置一个断点, 按下F5进入调试模式, 这时程序从断点处执行, 按下F10执行下一条语句, 如果这一条语句里面有函数的调用, 就可以按下F11进入该函数, 在函数内部按F10可以单步执行.

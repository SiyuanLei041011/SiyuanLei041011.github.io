集成电路版图设计详细复习笔记

第一章：版图设计基础与概念

1.1 版图的本质与重要性

1.1.1 版图的定义解析

版图（Layout） 是集成电路设计的物理实现，它是将电路逻辑描述转化为可以在硅片上制造的几何图形的过程。具体来说：

  • 几何层面：由不同工艺层（Layer）组成的二维图形集合
  • 电气层面：实现电路功能的导体、绝缘体和半导体区域
  • 物理层面：满足制造工艺约束的三维结构投影
  • 功能层面：确保电路性能、功耗和可靠性的空间布局
1.1.2 版图设计的核心目标
  1. 功能正确性：准确实现电路功能
  2. 性能优化：满足速度、功耗等性能指标
  3. 面积最小化：减少芯片成本
  4. 可制造性：符合工艺约束，提高良率
  5. 可靠性：确保长期稳定工作
1.1.3 版图与电路图的关系
  • 正向设计：电路图 → 版图（设计实现）
  • 反向验证：版图 → 电路图（一致性检查）

第二章：CMOS工艺与版图制造流程

2.1 CMOS工艺基础

2.1.1 基本材料与结构

衬底类型：

  • P型衬底：掺杂硼（B），多数载流子为空穴
  • N型衬底：掺杂磷（P）或砷（As），多数载流子为电子

典型工艺组合：

  • N阱CMOS：P型衬底，制作N阱容纳PMOS
  • P阱CMOS：N型衬底，制作P阱容纳NMOS
  • 双阱CMOS：轻掺杂衬底，同时制作N阱和P阱

5. 接触与互连

通孔（Via）：

  • 连接相邻金属层
  • 尺寸通常与接触孔相同
  • 材料：钨或铜

关键尺寸计算：

  3. 使用叉指结构：将宽晶体管拆分为多个窄指并联

5.1.3 应力工程

应力来源：

  1. 浅槽隔离应力：STI引起的压缩应力

  3. 金属化应力：金属与介质CTE不匹配
  4. 封装应力：芯片与封装材料CTE不匹配

应力影响：

版图对策：

  1. 增加有源区到STI距离：减小STI应力影响
  2. 对称布局：使应力分布对称
  3. 应力隔离：添加应力缓冲层
  4. 利用应力：有意引入应力提高性能
